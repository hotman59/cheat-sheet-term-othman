commande mv "move"

mv toto tata : pour renommer le fichier ou repertoire toto en tata

mv toto ~/Desktop : deplace le fichier ou le repertoire toto dans le repertoire desktop

mv toto ~/Desktop/tata : deplace le fichier ou document dans le repertoire Desktop et renomme toto en tat

commande cp "copy"

cp toto tata : fait copier tout les contenus de toto dans le fichier tata tout en preservant le fichier toto

commande rm "remove"

rm toto : supprime le fichier "toto"

rmdir : supprime un repertoire

commande cat

cat fichier_toto : affiche le fichier fichier_toto

cat -n fichier toto : affiche le fichier fichier_toto avec les numero de ligne

cat *.c: affiche tout les fichier avec .c
